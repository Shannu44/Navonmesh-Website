IT Events - Registration

Start Quest (QUIZ)

1. Two members make a team.
2. There will be multiple rounds, including sub-rounds.
3. Questions will focus on IT-related topics.
4. No electronic gadgets are allowed.

Fornax Code (Error Debugging)

1. Individual event.
2. Participants can choose any programming language like Python or C++.
3. No cheating or gadgets allowed.

Pictoris (Collage)

1. Two members make a team.
2. The collage should be on the theme of the First Indian Supercomputer.
3. A3 sheet, newspaper, glue, sketch pens, and scissors will be provided.

Orion’s Eye (Videography)

1. Individual event.
2. Participants are allowed to use any equipment.
3. No damaging of college property.
4. Topics will be given on the spot, and photos/videos should be taken within the campus.
